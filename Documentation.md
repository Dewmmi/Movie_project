Installation et Configuration
Extraction de l'archive : Placez le dossier MOVIE_PROJECT dans le répertoire de votre serveur local (C:/wamp64/www/movie_project).

Configuration de la base de données :

- Ouvrez phpMyAdmin.
- Créez une nouvelle base de données nommée project-movies.
- Allez dans l'onglet Importer et sélectionnez le fichier situé dans /sql/project-movies.sql.

Lancement : Accédez au site via l'adresse suivante : http://localhost/MOVIE_PROJECT/index.php

Technologies utilisées

1. Frontend : HTML5 et CSS pour l'interface utilisateur.
2. Backend : PHP procédural pour la logique serveur et la gestion des sessions.
3. Base de données : MySQL pour le stockage persistant des données.
