Movie Manager Project (PHP & MySQL)
A custom-built movie management system developed at SUPINFO Paris to demonstrate backend logic, database modeling, and session management using a local WampServer environment.
This application is a fully autonomous web platform designed to manage a movie database. Unlike projects using external APIs, I designed the entire relational database schema and implemented the PHP logic "from scratch" to master the fundamentals of web architecture.

- Backend: Procedural PHP (Server-side logic & Session management).
- Database: MySQL (Relational data storage).
- Server Environment: WampServer (Apache / MySQL).
- Frontend: HTML5 & CSS3 for a responsive user interface.

------------------------------------------------------------------------------------------------------------------------------
To set up the project locally, follow these steps:
------------------------------------------------------------------------------------------------------------------------------

- Extract the Archive: Place the MOVIE_PROJECT folder into your local server directory (e.g., C:/wamp64/www/movie_project).

Database Configuration:
- Open phpMyAdmin.
- Create a new database named project-movies.
- Go to the Import tab and select the SQL file located at /sql/project-movies.sql.
- Launch: Access the application in your browser at: http://localhost/MOVIE_PROJECT/index.php.
